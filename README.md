📊 Netflix Movie Data Analysis

This project focuses on analyzing a dataset of 9,000+ Netflix movies to uncover trends in genres, ratings, and popularity using Python.

🎯 Objective
Understand movie trends and audience preferences
Analyze ratings and popularity patterns
Generate insights for data-driven decision-making
🧹 Data Processing
Handled missing values and cleaned dataset
Processed data using Pandas and NumPy
📊 Exploratory Data Analysis (EDA)
Identified most frequent genres
Analyzed highest-rated movies
Compared popularity vs ratings
Studied year-wise movie trends
📈 Visualization
Created graphs using Matplotlib and Seaborn
Visualized genre distribution and popularity trends
🔍 Key Insights
Certain genres (like Drama) dominate the dataset
High popularity does not always mean high ratings
Recent years show an increase in movie releases
🛠️ Tools & Technologies
Python
Pandas
NumPy
Matplotlib
Seaborn
